Arkos automatically provides three categories of endpoints out of the box. This guide gives you a complete overview of what's available—no configuration required.

## Three Categories of Endpoints

### 1. CRUD Endpoints (From Prisma Models)
For every model in your Prisma schema, Arkos generates a complete RESTful API. The model name is converted to kebab-case and pluralized for the route.

**Example:** A `UserProfile` model becomes accessible at `/api/user-profiles`

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/[model]` | List records with filtering, pagination, sorting |
| POST | `/api/[model]` | Create one record |
| GET | `/api/[model]/:id` | Get one record |
| PATCH | `/api/[model]/:id` | Update one record |
| DELETE | `/api/[model]/:id` | Delete one record |
| POST | `/api/[model]/many` | Create multiple records |
| PATCH | `/api/[model]/many` | Update multiple records |
| DELETE | `/api/[model]/many` | Delete multiple records |

**Built-in capabilities:**
- Filtering, sorting, pagination via query parameters
- Field selection
- Full-text search on string fields
- Automatic relation handling

### 2. Authentication Endpoints
Built-in endpoints for user management and session handling.

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/login` | Authenticate user |
| POST | `/api/auth/signup` | Register new user |
| DELETE | `/api/auth/logout` | End session |
| POST | `/api/auth/update-password` | Change password |
| GET | `/api/users/me` | Get current user |
| PATCH | `/api/users/me` | Update current user |
| DELETE | `/api/users/me` | Delete current user |

### 3. File Upload Endpoints
Dedicated endpoints for handling file operations independently.

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/uploads/:fileType` | Upload files (images, videos, documents, files) |
| PATCH | `/api/uploads/:fileType/:fileName` | Replace existing file |
| DELETE | `/api/uploads/:fileType/:fileName` | Delete file |

`:fileType` accepts: `images`, `videos`, `documents`, or `files`

## How Generation Works

Arkos generates these endpoints by:
1. Reading your Prisma schema to understand your data models
2. Creating standardized controllers for each CRUD operation
3. Mounting them under your configured `globalPrefix` (default: `/api`)
